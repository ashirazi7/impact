Installing Packages

$ impact install "PackageName <version>"
$ impact install PackageName#<version>
$ impact install PackageName

Example

$ impact install PowerSystems
Installing version '0.2.0' of package 'PowerSystems'
  No information for package Complex, skipping
  No information for package ModelicaServices, skipping

Result

$ tree -L 1
.
├── Modelica 3.2
└── PowerSystems 0.2.0
